# Θέμα
Το θέμα του project είναι μία εικονική βιβλιοθήκη. Μπορούμε να δουμε τα διαθέσιμα
βιβλία με τις βασικές τους πληροφορίες (τίτλος, συγγραφέας και τιμή), να τα επεξεργαστούμε,
να τα διαγράψουμε ή να προσθέσουμε άν θέλουμε κάποια από αυτά σε μια δική μας λίστα.

# Τεχνικές Πληροφορίες
Η εφαρμογή είναι γραμμένη σε Java 17. Xρησιμοποιεί SpringBoot και τη βάση δεδομένων MySQL.
Περιλαμβάνει στρωματοποιημένες αρχιτεκτονικές ανάπτυξης με Repository, Service
Layer / Controllers στο Back-end και είναι Server Side Rendering με Thymeleaf
στο Front-end.

# Πως τρέχει και οδηγίες για setup
Μπορούμε να το τρέξουμε απλά με run του class LibappApplication.java.
Άλλος τρόπος είναι από Terminal με "gradlew clean build" και στη συνέχεια
"java -jar ./build/libs/libapp.jar"

Πριν τρέξει το πρόγραμμα πρέπει να υπάρχει μια κενή βάση δεδομένων MySQL στο pc
και ένας χρήστης με όλα τα δικαιώματα σε αυτή. Τα στοιχεία για τη σύνδεση με τη
ΒΔ βρίσκονται στο αρχείο application.properties. Αφού τρέξει την πρώτη φορά
και δημιουργηθούν οι πίνακες στη ΒΔ, κάνουμε uncomment τις εντολές στις γραμμές
26, 27 στο application.properties για να εισάγουμε μερικά βιβλία στη ΒΔ, τρέχουμε
το πρόγραμμα μία ακόμα φορά και μετά βάζουμε πάλι comment.

Αφού έχουν γίνει όλα τα παραπάνω η εφαρμογή ξεκινάει στο http://localhost:8080/.

Το πρώτο πράγμα που πρέπει να γίνει είναι register και στη συνέχεια login με τον
χρήστη που φτιάξαμε, ώστε να έχουμε πρόσβαση στην εφαρμογή.
